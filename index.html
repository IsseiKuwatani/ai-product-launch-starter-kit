ようこそ、AIプロダクト立ち上げラボへ

Claude Code / Codex を使って、非エンジニアでもAI Webサービスの最初の形を作り始めるための実用セットです。まずは自分のPCで動く簡単なサービスを作り、必要に応じて機能を広げていく方針です。マニュアルを見るだけでなく、テンプレート内の /setup から初期設定まで進められます。

キットを開く 自分のテーマで相談する

※ URLに ?name=お名前 が付いている場合は、ページ上部にお名前を表示します。

立ち上げ後の最短ルート
$ cd template/next-ai-product-starter
$ npm install
$ claude

「ダウンロードして終わり」ではなく、立ち上げまで進める設計

テンプレートを開いた後は、Claude Code内で /setup を実行 すればOKです。追加の資料を探し回らなくても、質問に答えながら、まずは自分のPCで動く形まで進められます。資料はこのページから確認用に見られるように残しています。
1. まず見る

マニュアル・プロンプト・ワークシートをブラウザで確認。

2. テンプレを開く

Next.jsスターターを自分のPC/リポジトリで開く。

3. /setup

Claude Codeが質問しながら1画面MVPに整える。

4. 相談へ

詰まったら自分のテーマで個別相談へ進む。

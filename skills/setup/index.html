Setup Skill — Next.js AI Product Starter

実際にClaude Codeから使う本体は template/next-ai-product-starter/.claude/skills/setup/SKILL.md に入っています。このページはブラウザ確認用です。

使う場面

ユーザーがテンプレートをClaude Codeで開き、/setup または「セットアップして」と依頼したときに使います。目的は、テンプレートをその場で起動し、事業アイデアに合わせて最小MVPへ整えることです。

1. 現在状態の確認

pwd
ls
node -v
npm -v

主要ファイルとして package.json、app/page.tsx、components/ProductForm.tsx、app/globals.css、README.md を確認します。

2. 依存関係のインストール

npm install

3. ユーザーに3つだけ質問する

  1. 作りたいサービス名は何ですか？
  2. 誰のどんな課題を解決しますか？
  3. ユーザーに何を入力してもらい、どんな結果を返したいですか？

4. 1画面MVPへ編集する

  • app/page.tsx: ヒーローコピー、説明文、CTA、利用ステップ
  • components/ProductForm.tsx: 入力項目、ボタン文言、モック結果
  • app/globals.css: 必要に応じて色・余白・スマホ表示

1回目はAI API接続まで広げすぎず、まずモック結果で「体験」が伝わる状態にします。APIキーや秘密情報はコードに直書きしません。

5. 起動して確認する

npm run dev
  • http://localhost:3000 が開く
  • スマホ幅でも崩れない
  • フォーム入力 → 結果表示まで動く
  • CTA文言が具体的

6. 次の改善案を3つだけ出す

  1. AI APIを接続して本物の結果生成にする
  2. 相談・予約・LINE登録などの導線を追加する
  3. 利用者の入力内容を保存するDBを追加する

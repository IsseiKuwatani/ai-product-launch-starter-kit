# AI Webサービス制作スターターキット

AIプロダクト立ち上げラボ by ツギノメ の無料配布キットです。

## このキットでできること

非エンジニアの方が、Claude Code / Codex を使って「人に見せられるAI Webサービスの最初の形」を作り始めるためのスターターセットです。

## 5つの配布物

1. [Claude Code / Codex 導入マニュアル](docs/01-claude-code-codex-setup-manual.md)
2. [開発プロンプト集](prompts/02-development-prompts.md)
3. [skills・エージェント設計集](skills/03-skills-and-agents.md)
4. [AIプロダクト企画・要件定義ワークシート](docs/04-product-planning-worksheet.md)
5. [制作テンプレート: Next.js AI Product Starter](template/next-ai-product-starter/)

## おすすめの使い方

1. まず `docs/04-product-planning-worksheet.md` で作りたいものを1ページに整理する
2. `docs/01-claude-code-codex-setup-manual.md` に沿って開発環境を整える
3. `template/next-ai-product-starter/` を自分のPCにコピーする
4. `prompts/02-development-prompts.md` のプロンプトをClaude Code / Codexへ貼る
5. `skills/03-skills-and-agents.md` を使って、企画・実装・レビューを分担させる

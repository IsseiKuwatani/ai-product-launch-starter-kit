結論：今契約しているAIで始めてOKです。Claudeを使っている人はClaude Code、ChatGPT / OpenAIを使っている人はCodexから始めてください。
選び方へ戻る/setup説明へプロンプト集へ

まず、どれを選べばいい？

Mac の人

Mac版マニュアルを見ます。Claude契約中ならClaude Code Mac、ChatGPT / OpenAI契約中ならCodex Macへ。

Windows の人

Windows版マニュアルを見ます。PowerShellなどの言葉が出ても、まずはWindows用ページだけ見ればOKです。

AI選びで迷う人

新しく契約を増やすより、今契約しているAIでやる方がいいです。ツールが違っても、最初のゴールは同じです。

プロンプト集と /setup skill の違い

項目プロンプト集/setup skill
一言でいうと順番にコピーしてAIに貼る「台本」Claude Code向けの「初回案内係」
使う場所Claude Code / Codex などの入力欄Claude Codeでテンプレートを開いた後の入力欄
向いている人中身を見ながら、自分のペースで進めたい人最初の1画面MVPまで迷わず進めたい人
Codexの場合ページURLやファイルパスを渡して「この手順で進めて」と頼むslashとして実行できなくても、setup説明ページやskillファイルを読ませて同じ方針で進められる

おすすめルート

Claude Codeルート

導入マニュアル → テンプレートを開く → /setup → ブラウザ確認 → 必要に応じてプロンプト集

Codexルート

導入マニュアル → テンプレートを開く → ヘルプ/プロンプト集のリンクやファイルパスをCodexに渡す → ブラウザ確認 → 必要に応じて保存/AI機能

Codexに最初に渡す文

Codexでテンプレートを開いたら、まず以下のように依頼してください。

このフォルダでAI WebサービスのMVPを作ります。
docs/help.html と prompts/02-development-prompts.md を読んで、
まず1画面MVPを作る手順で進めてください。
必要なら skills/setup/SKILL.md も参考にしてください。
いきなり大きく作らず、入力フォームと結果表示までを小さく作ってください。

FAQ

Q. Claude CodeとCodexはどちらがいいですか？

A. 原則は 今契約しているAIでやる方がいい です。Claudeを契約中ならClaude Code、ChatGPT / OpenAIを契約中ならCodex。両方を新しく契約する必要はありません。

Q. 最初に見るのはプロンプト集ですか？ /setup ですか？

A. Claude Codeで進めるなら、最初は /setup がおすすめです。Codexで進める、または手順を見ながら自分で調整したい場合はプロンプト集を上から貼ってください。

Q. プロンプト集と /setup は両方やる必要がありますか？

A. 必須ではありません。/setup で最初の形を作り、その後の改善でプロンプト集を使う、という順番が分かりやすいです。

Q. /setup やヘルプはCodexでも使えますか？

A. はい、考え方としては使えます。Claude Codeのように /setup slashとして動かない場合でも、Codexに docs/help.html や prompts/02-development-prompts.md、skills/setup/SKILL.md のファイルパスを渡して「この手順を読んで進めて」と頼めばOKです。

Q. MacとWindowsのどちらのマニュアルを見ればいいですか？

A. 自分のPCに合わせます。MacならMac版、WindowsならWindows版です。

Q. ターミナルやコマンドが怖いです。

A. コピーして貼るだけの行を使います。意味が不安なときは、AIに「今から実行するコマンドの意味を1行で説明して」と聞いてから実行してください。

Q. AI APIキーは最初から必要ですか？

A. 不要です。最初はAI接続なしのモック結果で体験を確認します。「入力する → 結果が出る」流れができてから、必要な場合だけ追加します。

Q. どこまでできたら初回セットアップ完了ですか？

A. ブラウザで自分のPC上のページが開き、「入力する → 結果が表示される」まで確認できたら完了です。公開、ログイン、決済は次の段階でOKです。

# AIプロダクト企画・要件定義ワークシート

このシートを埋めると、Claude Code / Codex に渡しやすい仕様になります。

## 1. サービス名

例: AI営業メール診断、整体院向け予約ヒアリングAI、採用面談質問メーカー

```txt
サービス名:
```

## 2. 誰のためのサービスか

```txt
対象ユーザー:
職種/属性:
今困っていること:
```

## 3. 解決したい課題

```txt
ユーザーは何に困っているか:
なぜ今それが問題か:
解決できると何が嬉しいか:
```

## 4. 最初に作るAI機能

```txt
入力するもの:
AIが考えること:
出力するもの:
```

例:

```txt
入力: 業種、悩み、現在の施策
AIが考えること: 課題分類、改善案、優先順位
出力: 改善提案3つ、最初の行動、相談導線
```

## 5. 必要な画面

```txt
1. トップページ:
2. 入力フォーム:
3. 結果ページ:
4. 相談/問い合わせページ:
```

## 6. MVPで作らないこと

最初から全部作らないために、作らないことを決めます。

```txt
今は作らない機能:
理由:
```

## 7. CTA

```txt
最終的にユーザーにしてほしい行動:
ボタン文言:
リンク先:
```

## 8. AIへの実装依頼文

以下をコピペして使ってください。

```md
以下のAI WebサービスをNext.jsでMVP実装したいです。

# サービス名
[サービス名]

# 対象ユーザー
[対象ユーザー]

# 解決する課題
[課題]

# AI機能
入力: [入力]
処理: [AIが考えること]
出力: [出力]

# 必要画面
- トップページ
- 入力フォーム
- 結果ページ
- 相談/問い合わせ導線

# MVPで作らないこと
[作らないこと]

まず、実装計画とファイル構成を提案してください。
その後、トップページと入力フォームから実装してください。
```

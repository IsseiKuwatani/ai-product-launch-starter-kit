# /setup — Next.js AI Product Starter

`.claude/skills/setup/SKILL.md` の手順に沿って、このプロジェクトの初期セットアップを進めてください。

まず現在状態を確認し、必要なら `npm install` を実行してください。
その後、ユーザーに以下の3点だけ確認して、`app/page.tsx` と `components/ProductForm.tsx` を最小MVPに合わせて編集してください。

1. 作りたいサービス名
2. 誰のどんな課題を解決するか
3. ユーザーに何を入力してもらい、どんな結果を返すか

完了条件:

- ローカルで `npm run dev` できる
- フォーム入力からモック結果表示まで動く
- 画面コピーがユーザーのテーマに置き換わっている
- 次の改善案を3つだけ提案する

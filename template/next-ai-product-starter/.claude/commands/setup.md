# /setup — Next.js AI Product Starter

`.claude/skills/setup/SKILL.md` の手順に沿って、このプロジェクトの初期セットアップを進めてください。

重要方針:

- 外部サービス、本格的なデータベース、ログイン、決済は、ユーザーが明示的に依頼するまで導入しない
- まず `npm run dev` でローカルPC上で動く状態を作る
- 初期版はモック結果で「入力 → 結果表示」の体験を確認する
- 保存が必要な場合は、まず localStorage / JSONダウンロード / CSVダウンロードで対応する

まず現在状態を確認し、必要なら `npm install` を実行してください。
その後、ユーザーに以下の5点だけ確認して、`app/page.tsx` と `components/ProductForm.tsx` を最小MVPに合わせて編集してください。

1. 作りたいサービス名
2. 誰のどんな課題を解決するか
3. ユーザーに何を入力してもらうか
4. どんな結果を返したいか
5. 最後にユーザーに何をしてほしいか

完了条件:

- ローカルで `npm run dev` できる
- フォーム入力からモック結果表示まで動く
- 画面コピーがユーザーのテーマに置き換わっている
- 空欄時の案内がある
- 本格的なデータベース、ログイン、決済などが勝手に追加されていない
- 次の改善案を3つだけ提案する
